# Sobre Git

Git é um sistema de controle de versão distribuído. Com esse sistema, é possível recuperar versões anteriores de arquivos.

O Git funciona com repositórios - diretórios com os arquivos rastreados e uma pasta chamada _.git_. Essa pasta armazena todas as informações necessárias para o controle de versão.

Cada arquivo de um repositório pode estar em um de três estágios:

- No estágio _working_, o arquivo é uma cópia de uma versão do repositório e pode ser editado;
- No estágio _staging_, o arquivo está pronto para ter suas mudanças confirmadas em uma versão do repositório;
- No estágio _repository_, o arquivo tem suas mudanças confirmadas em uma versão do repositório.

Repositórios podem ser manipulados através de interfaces de linha de comando.
